# layou_user_avatar Example App

This example demonstrates the usage of the `layou_user_avatar` package.

## Setup

1. Configure Firebase for your project:
   ```bash
   flutterfire configure
   ```

2. Enable Firebase Authentication and Firebase Storage in the Firebase Console.

3. Run the app:
   ```bash
   flutter run
   ```

## Features Demonstrated

- **Avatar Editor (Vertical Layout)**: Avatar display with upload and delete buttons below.
- **Avatar Editor (Horizontal Layout)**: Avatar display with buttons on the right side.
- **Avatar Editor (Overlay Layout)**: Avatar display with an edit badge overlay.
- **Read-Only Avatar Display**: Simple avatar display without edit functionality.
- **Multiple Sizes**: Demonstrating avatars at different sizes (32px, 48px, 64px, 96px).
